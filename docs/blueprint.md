# **App Name**: Excel Insights Extractor

## Core Features:

- File Upload and Parsing: Upload an Excel file, parse its data, and display the column headers for selection.
- Column Selection and Configuration: Choose specific columns to include in the summary, set constant values for filtering, and define a date range.
- AI-Powered Summary Generation: Use AI to analyze the selected columns and generate a concise summary of the relevant information based on filter settings; The LLM acts as a tool and reasons to choose appropriate info.
- Downloadable Summary: Allow users to download the generated summary as an Excel (.xlsx) file.

## Style Guidelines:

- Primary color: Deep blue (#3F51B5) to evoke a sense of professionalism and trustworthiness.
- Background color: Light blue (#E8EAF6) to create a calm, unobtrusive backdrop.
- Accent color: Purple (#9575CD) for interactive elements and calls to action.
- Body and headline font: 'Inter' sans-serif for clear, readable text.
- Use clean, professional icons to represent file types, actions (upload, download), and settings.
- A clear, structured layout with a prominent upload area, configurable selection and filtering controls, and a well-defined summary display area.
- Subtle animations for file upload progress, summary generation, and download confirmation.
